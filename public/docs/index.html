Project Euler JavaScript solutions

This is a collection of solutions to Project Euler problems using JavaScript. This is a work in progress.

Contributing

Feel free to submit pull requests with solutions! In lieu of a formal styleguide, take care to maintain the existing coding style and lint your changes using grunt.

Run locally

Requires Express and Jade

$ git clone git://github.com/Fiveisprime/euler.nodester.git
$ cd euler.nodester
$ npm install express
$ npm install jade
$ node app.js

Once the app is running, navigate to http://localhost:20109 to see the result.
